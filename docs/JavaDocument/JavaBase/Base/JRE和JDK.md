#### 2.35 JRE和JDK

##### 2.35.1 JRE(Java Runtime Enviroment)

是Java程序运行时环境，包含JVM和运行时所需要的核心类库。
我们想要==运行==一个已有的Java程序，那么只需要安装JRE即可。

##### 2.35.2 JDK(Java Developemtn Kit)

是Java程序开发工具包，包含JRE和开发人员使用的工具。
其中的开发工具：编译工具(javac.exe)和运行工具(java.exe)
