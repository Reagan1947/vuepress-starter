#### 2.53 内部类

内部类：就是在一个类中定义一个类。举例：在一个类A的内部定义一个类B，类B就被称为内部类。

内部类的定义格式

- 格式：

  ```java
  public class 类名 {
      修饰符 class 类名 {
          
      }
  }
  ```

- 范例：

  ```java
  public class Outer {
      public class Inner {
          
      }
  }
  ```

内部类的访问特点

- 内部类可以直接访问外部类的成员，包括私有
- 外部类要访问内部类的成员，必须创建对象
