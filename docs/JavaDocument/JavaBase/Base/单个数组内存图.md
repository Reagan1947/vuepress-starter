## 1.2 参考代码

```java
public class main
{
    public static void main(String[] args) {
        int[] arr = new int[3];
        // 输出数组名及元素
        System.out.println(arr);

        System.out.println(arr[0]);
        System.out.println(arr[1]);
        System.out.println(arr[2]);
        
        // 给数组中的元素赋值
        arr[0] = 100;
        arr[2] = 200;
        
        // 再次输出数组名及元素
        System.out.println(arr);
        System.out.println(arr[0]);
        System.out.println(arr[1]);
        System.out.println(arr[2]);
    }
}
```
