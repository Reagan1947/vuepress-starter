# 数组定义格式

## 1.1 数组概述

- 一次性声明「大量」的用于存储数据的变量
- 要存储的数据通常是「同类型的数据」，例如：考试成绩

## 1.2 什么是数组

数组 (array) 是一种用于存储「多个」「相同类型」数据的存储模型

## 1.1 数组的定义格式

- 格式一：数据类型[ ]   变量名

  范例：`int[] arr`

  定义了一个 int 类型的数组，数组名是 arr

- 格式二： 数据类型   变量名[ ]

  范例：`int arr[]`

  定义一个 int 类型的变量，变量名是 arr 数组

## 1.2 参考代码

```java
public class main
{
    public static void main(String[] args) {
        int[] arr = new int[3];
        /*
            左边：
            int：说明数组中的元素类型是int类型
            []: 说明这是一个数组
            arr：这是数组的名称

            右边：
            new：为数组申请内存空间
            int：说明数组中的元素是int类型
            []：说明这是一个数组
            3：数组长度，其实就是数组中元素的个数
         */
    }
}

```
