#### 2.8 数组定义格式

##### 2.8.1 数组的定义格式

- 格式一：数据类型[ ]   变量名

  范例：int[ ]    arr

  定义了一个int类型的数组，数组名是arr

- 格式二： 数据类型   变量名[ ]

  范例：int    arr[ ]

  定义一个int类型的变量，变量名是arr数组

```java
public class main
{
    public static void main(String[] args) {
        int[] arr = new int[3];
        /*
            左边：
            int：说明数组中的元素类型是int类型
            []: 说明这是一个数组
            arr：这是数组的名称

            右边：
            new：为数组申请内存空间
            int：说明数组中的元素是int类型
            []：说明这是一个数组
            3：数组长度，其实就是数组中元素的个数
         */
    }
}

```
