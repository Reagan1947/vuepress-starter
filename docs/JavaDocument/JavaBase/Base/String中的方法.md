#### 2.28 String中的方法

| 方法名                                 | 说明                             |
| -------------------------------------- | -------------------------------- |
| public boolean equals(Object anObject) | 比较字符串的内容，严格区分大小写 |
| public char charAt(int index)          | 返回执行索引处的char值           |
| public int length()                    | 返回字符串的长度                 |
