#### 2.59 System

##### 2.59.1 System类概述

System类包含几个有用的类字段和方法，它不能被实例化

##### 2.59.2 System类的常用方法

| 方法名                                 | 说明                                       |
| -------------------------------------- | ------------------------------------------ |
| public static void exit(int status)    | 终止当前运行的Java虚拟机，非零表示异常终止 |
| public static long currentTimeMillis() | 返回当前时间(以毫秒为单位)                 |
