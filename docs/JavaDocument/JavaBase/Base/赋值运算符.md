# 赋值运算符

## 1.1 概述

| 符号 | 作用       | 说明                                              |
| ---- | ---------- | ------------------------------------------------- |
| =    | 赋值       | `a = 10`, 将 10 赋值给变量 a                      |
| +=   | 加后赋值   | `a += b`, 将 a + b 的值赋值给 a                   |
| -=   | 减后赋值   | `a -= b`,  将 a - b 的值赋值给 a                  |
| *=   | 乘后赋值   | `a *= b`, 将 a * b 的值赋值给 a                   |
| /=   | 除后赋值   | `a /= b`, 将 a / b 的值赋值给 a (a 除以 b 的商)   |
| %=   | 取余后赋值 | `a %= b`, 将 a % b 的值赋值给 a (a 除以 b 的余数) |
