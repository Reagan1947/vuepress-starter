# 赋值运算符

## 1.1 概述

| 符号 | 作用       | 说明                                       |
| ---- | ---------- | ------------------------------------------ |
| =    | 赋值       | a = 10, 将10赋值给变量a                    |
| +=   | 加后赋值   | a += b, 将a + b的值赋值给a                 |
| -=   | 减后赋值   | a -= b,  将a - b的值赋值给a                |
| *=   | 乘后赋值   | a *= b, 将a * b的值赋值给a                 |
| /=   | 除后赋值   | a /= b, 将a / b的值赋值给a(a 除以 b的商)   |
| %=   | 取余后赋值 | a %= b, 将a % b的值赋值给a(a 除以 b的余数) |
