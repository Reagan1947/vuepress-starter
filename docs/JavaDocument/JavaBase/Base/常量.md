# 常量

##  1.1常量概述

常量：在程序运行过程中，其值不可以发生改变的量。

| 常量类型   | 说明                 | 举例                    |
| ---------- | -------------------- | ----------------------- |
| 字符串常量 | 用双引号括起来的内容 | “Hello World”           |
| 整数常量   | 不带小数的数字       | 666，-88                |
| 小数常量   | 带小数的数字         | 13.14， -5.21           |
| 字符常量   | 用单引号括起来的内容 | ’A‘, '0', '我'          |
| 布尔常量   | 布尔值，表示真假     | 只有两个值：true，false |
| 空常量     | 一个特殊的值，空值   | 值是：null              |

\* 空常量(null)不能直接输出
