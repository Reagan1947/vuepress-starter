# 关系运算符

## 1.1 概述

| 符号 | 范例   | 说明(成立为 true，不成立为 false) |
| ---- | ------ | --------------------------------- |
| ==   | a == b | 判断 a 和 b 是否相等              |
| !=   | a != b | 判断 a 和 b 是否不相等            |
| >    | a > b  | 判断 a 是否大于 b                 |
| >=   | a >= b | 判断 a 是否大于等于 b             |
| <    | a < b  | 判断 a 是否小于 b                 |
| <=   | a <= b | 判断 a 是否小于等于 b             |

\* 注意事项：
关系运算符的结果都是boolean类型，要么是true要么是false。

```java
public class main
{
    public static void main(String[] args)
    {
        // 定义变量
        int i = 10;
        int j = 20;
        int k = 10;

        // ==
        System.out.println(i == j);
        System.out.println(i == k);
        System.out.println("-------");

        // !=
        System.out.println(i != j);
        System.out.println(i != k);
        System.out.println("-------");

        // >
        System.out.println(i > j);
        System.out.println(i > k);
        System.out.println("-------");

        // >=
        System.out.println(i >= j);
        System.out.println(i >= k);
        System.out.println("-------");

        // Mistake ==
        System.out.println(i = j);
    }
}
```
