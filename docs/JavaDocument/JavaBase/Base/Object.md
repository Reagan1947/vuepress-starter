#### 2.60 Object

##### 2.60.1 Object 类的概述

Object类是类层次结构的根，每个类都可以将Object类作为超类。所有类都直接或间接的继承自该类。

构造方法：public Object()

回想面向对象中，为什么说子类的构造方法默认访问的是父类的无参构造方法？
因为它们的顶级父类只有无参构造方法

##### 2.60.2 Object类的常用方法

| 方法名                            | 说明                                                       |
| --------------------------------- | ---------------------------------------------------------- |
| public String toString()          | 返回对象的字符串表示形式。建议所有子类重写该方法，自动生成 |
| public boolean equals(Object obj) | 比较对象是否相同。默认比较地址，重写可以比较内容，自动生成 |
